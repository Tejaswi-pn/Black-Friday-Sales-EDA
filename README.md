# 🛒 Black Friday Sales - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Black Friday Sales dataset to uncover customer purchasing behavior and identify business insights.

The analysis includes data cleaning, visualization, feature exploration, and interpretation of customer demographics, product categories, and purchasing patterns.

---

# 📂 Dataset

**Dataset Name:** Black Friday Sales Dataset

The dataset contains customer purchase records collected during Black Friday sales.

Features include:

- User_ID
- Product_ID
- Gender
- Age
- Occupation
- City_Category
- Stay_In_Current_City_Years
- Marital_Status
- Product_Category_1
- Product_Category_2
- Product_Category_3
- Purchase

---

# 🎯 Objectives

- Understand customer purchasing behavior
- Identify the highest spending customer groups
- Analyze purchasing trends across age groups
- Explore gender-based purchasing patterns
- Study product category popularity
- Discover business insights using visualization

- Customers aged 26–35 are among the highest purchasers.
